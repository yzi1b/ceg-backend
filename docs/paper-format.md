# 试卷格式文档

## 概述

试卷以 JSON 格式存储，包含试卷元信息、题目列表和参考答案列表。

## 顶层结构

```json
{
  "title": "试卷标题",
  "description": "试卷说明（可选）",
  "questions": [ ... ],
  "answers": [ ... ]
}
```

| 字段 | 类型 | 必需 | 说明                          |
|------|------|----|-----------------------------|
| `title` | string | 否  | 试卷标题，可选                     |
| `description` | string | 否  | 试卷说明或备注，暂未使用                |
| `questions` | array | 是  | 题目对象数组，不能为空                 |
| `answers` | array | 是  | 参考答案数组，长度必须与 `questions` 一致 |

`answers` 中每个元素对应 `questions` 中同一下标的题目的答案。各题型的答案格式见下文。

---

## 题型

### 1. 单选题 (obj.ssq)

**类型标识：** `obj.ssq`

**题目格式：**

```json
{
  "type": "obj.ssq",
  "content": "题目标题/题干",
  "options": ["选项A", "选项B", "选项C", "选项D"],
  "score": 4
}
```

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `type` | string | 是 | 固定为 `"obj.ssq"` |
| `content` | string | 是 | 题干，不能为空 |
| `options` | array[string] | 是 | 选项列表，至少 2 个选项，每个不能为空 |
| `score` | integer | 是 | 分值，必须为正整数 |

**答案格式：** 正确选项的索引（从 0 开始），类型为 integer。

```json
// 答案示例：选择第 4 个选项
3
```

**评分规则：** 与标准答案完全一致得满分，否则 0 分。

**完整示例：**

```json
{
  "title": "示例试卷",
  "description": "单选题示例",
  "questions": [
    {
      "type": "obj.ssq",
      "content": "下列对于机械动力进度取得方式的说法中，正确的是",
      "options": [
        "安山时代：首次获得安山合金",
        "坚如磐石：制造一个安山机壳",
        "自动驾驶：使用列车时刻表指挥列车司机",
        "滚滚而来：看见物品滚过多个连续的分液池"
      ],
      "score": 4
    }
  ],
  "answers": [3]
}
```

### 2. 多选题 (obj.msq)

**类型标识：** `obj.msq`

**题目格式：**

```json
{
  "type": "obj.msq",
  "content": "题目标题/题干",
  "options": ["选项A", "选项B", "选项C", "选项D"],
  "strict": true,
  "score": 6,
  "partial": 2
}
```

| 字段 | 类型 | 必需 | 说明                                                 |
|------|------|------|----------------------------------------------------|
| `type` | string | 是 | 固定为 `"obj.msq"`                                    |
| `content` | string | 是 | 题干，不能为空                                            |
| `options` | array[string] | 是 | 选项列表，至少 3 个选项，每个不能为空                               |
| `strict` | boolean | 是 | 是否是严格多选题，如果为 true，将报告考生为多选题；如果为 false，将报告考生为不定项选择题 |
| `score` | integer | 是 | 满分，必须为正整数                                          |
| `partial` | integer | 是 | 部分正确时得分，范围为 [0, score]                             |

**答案格式：** 正确选项索引数组（从 0 开始），无重复、不排序要求。

```json
// 答案示例：选择第 1 个和第 3 个选项
[0, 2]
```

**评分规则：**
- 答案中包含任何错误选项（不在标准答案中）→ 0 分
- 答案有重复项 → 0 分
- 答案与标准答案完全匹配（长度和元素一致）→ 满分
- 答案中所有选项均正确，但未选全 → 得 `partial` 分

**完整示例：**

```json
{
  "title": "示例试卷",
  "description": "多选题示例",
  "questions": [
    {
      "type": "obj.msq",
      "content": "下列物品中仅需同种机壳就能合成的有",
      "options": [
        "十字齿轮箱，锁链传送箱，离合器，转速控制器",
        "动力锯，动力钻头，动力搅拌器，动力冲压机",
        "机械手，动力臂，存量转信器，智能侦测器",
        "分液池，软管滑轮，注液器，移动式流体接口"
      ],
      "strict": true,
      "score": 6,
      "partial": 2
    }
  ],
  "answers": [[0, 2]]
}
```

### 3. 客观填空题 (obj.fib)

**类型标识：** `obj.fib`

**题目格式：**

```json
{
  "type": "obj.fib",
  "content": "题目标题/题干",
  "blanks": [
    ["描述1", 3],
    ["描述2", 2]
  ]
}
```

| 字段 | 类型 | 必需 | 说明                      |
|------|------|------|-------------------------|
| `type` | string | 是 | 固定为 `"obj.fib"`         |
| `content` | string | 是 | 题干，不能为空                 |
| `blanks` | array[array] | 是 | 填空项数组，每个元素为 `[空描述, 分值]` |
| `blanks[i][0]` | string | 是 | 该空的参考答案                 |
| `blanks[i][1]` | integer | 是 | 该空分值，必须为正整数             |

**答案格式：** 字符串数组，长度必须与 `blanks` 一致。

```json
// 答案示例
["标准答案1", "标准答案2"]
```

**评分规则：** 逐空对比，答案字符串与 `blanks` 中对应标准答案完全一致（`===`）时，获得该空分值；否则该空 0 分。总分为各空得分之和。

**完整示例：**

```json
{
  "title": "示例试卷",
  "description": "客观填空题示例",
  "questions": [
    {
      "type": "obj.fib",
      "content": "天空是__1__色的，草是__2__色的。",
      "blanks": [
        ["1.", 3],
        ["2.", 2]
      ]
    }
  ],
  "answers": [["蓝", "绿"]]
}
```

### 4. 主观填空题 (sbj.fib)

**类型标识：** `sbj.fib`

**题目格式：**

```json
{
  "type": "sbj.fib",
  "content": "题目标题/题干",
  "blanks": [
    ["描述1", 3],
    ["描述2", 2]
  ]
}
```

| 字段 | 类型 | 必需 | 说明                      |
|------|------|------|-------------------------|
| `type` | string | 是 | 固定为 `"sbj.fib"`         |
| `content` | string | 是 | 题干，不能为空                 |
| `blanks` | array[array] | 是 | 填空项数组，每个元素为 `[空描述, 分值]` |
| `blanks[i][0]` | string | 是 | 该空参考评分依据                |
| `blanks[i][1]` | integer | 是 | 该空分值，必须为正整数             |

**答案格式：** 与客观填空题一致，字符串数组，长度必须与 `blanks` 一致。

```json
["参考答案1", "参考答案2"]
```

**评分规则：** 主观题，`score()` 始终返回 0，需人工阅卷评分。

**完整示例：**

```json
{
  "title": "示例试卷",
  "description": "主观填空题示例",
  "questions": [
    {
      "type": "sbj.fib",
      "content": "鲁迅出生于__1__年，代表作是__2__。",
      "blanks": [
        ["1.", 3],
        ["2.", 2]
      ]
    }
  ],
  "answers": [["1881", "狂人日记"]]
}
```

---

### 5. 简答题 (sbj.saq)

**类型标识：** `sbj.saq`

**题目格式：**

```json
{
  "type": "sbj.saq",
  "content": "题目标题/题干",
  "score": 10
}
```

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `type` | string | 是 | 固定为 `"sbj.saq"` |
| `content` | string | 是 | 题干，不能为空 |
| `score` | integer | 是 | 分值，必须为正整数；兼容字段 `full` |

**答案格式：** 字符串，为参考答案。

```json
"这是参考答案"
```

**评分规则：** 主观题，`score()` 始终返回 0，需人工阅卷评分。

**完整示例：**

```json
{
  "title": "示例试卷",
  "description": "简答题示例",
  "questions": [
    {
      "type": "sbj.saq",
      "content": "请简述牛顿第一定律的内容。",
      "score": 10
    }
  ],
  "answers": ["一切物体在没有受到力的作用时，总保持静止状态或匀速直线运动状态。"]
}
```

## 题型对照表

| 类型标识 | 题型 | 客观/主观 | 自动评分 |
|----------|------|-----------|----------|
| `obj.ssq` | 单选题 | 客观 | 是 |
| `obj.msq` | 多选题 | 客观 | 是 |
| `obj.fib` | 客观填空题 | 客观 | 是 |
| `sbj.fib` | 主观填空题 | 主观 | 否 |
| `sbj.saq` | 简答题 | 主观 | 否 |

## 约束规则

1. `questions` 数组不能为空，且 `questions` 与 `answers` 长度必须相等。
2. 每道题必须有 `type` 字段，且值必须是上述 5 种类型标识之一。
3. 题目构造过程中若解析失败，会抛出异常：
   - 题目数据不合法 → 错误 `question_phase_error`
   - 答案数据不合法 → 错误 `answer_invalid`
